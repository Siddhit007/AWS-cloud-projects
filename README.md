# ☁️ AWS Cloud Projects Portfolio

This repository showcases hands-on projects built on AWS, focusing on real-world cloud architecture, scalability, and DevOps practices.

## 🎯 Goal

To build practical cloud projects and gain real-world AWS experience.
1. I created a basic web hosting setup using AWS EC2 where I launched an instance, configured security groups to allow HTTP traffic, and installed Nginx to host a static website.
2. I also extended it by adding multiple pages and later replaced Nginx with Apache to understand different web servers.
3. Additionally, I implemented static website hosting using S3, where I configured bucket policies to allow public access.
4. Implemented end-to-end CI/CD lab by integrating Jenkins controller on WSL with AWS EC2 remote agent and Apache Tomcat deployment server using SSH-based automation.
